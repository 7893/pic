## ✅ 已配置

```
触发时间：每小时整点（0 * * * *）
下载数量：1500 张/次
预计：36,000 张/天
```

---

## ⚠️ 重要提醒

### Unsplash API 限制
- **免费版**：50 次/小时
- **当前需求**：50 次/小时（1500 ÷ 30）
- **状态**：⚠️ 刚好达到限制，建议升级或降低频率

### 建议调整

#### 选项1：降低频率（推荐）
```toml
# 每2小时一次
crons = ["0 */2 * * *"]
```
- 下载：750 张/次，18,000 张/天
- API：25 次/小时

#### 选项2：减少数量
```javascript
// 每次500张
body: JSON.stringify({ count: 500 })
```
- 下载：500 张/次，12,000 张/天
- API：17 次/小时

#### 选项3：错峰下载
```toml
# 每4小时一次
crons = ["0 */4 * * *"]
```
- 下载：1500 张/次，9,000 张/天
- API：50 次/4小时

## 🔧 快速命令

### 查看日志
```bash
npx wrangler tail
```

### 查看统计
```bash
curl "https://pic.53.workers.dev/api/category-stats"
```

### 禁用 Cron
编辑 `wrangler.toml`，注释掉：
```toml
# [triggers]
# crons = ["0 * * * *"]
```
然后重新部署：
```bash
npx wrangler deploy
```

## 📅 下次触发时间

Cron 将在下一个整点触发（例如：19:00, 20:00, 21:00...）

查看日志确认：
```bash
npx wrangler tail --format=pretty
```
